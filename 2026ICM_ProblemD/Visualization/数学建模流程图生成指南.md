# 数学建模流程图生成指南

本指南提供了针对数学建模论文中不同场景下生成高质量流程图的通用提示词模板。您只需将模板中的占位符替换为您的具体内容，即可生成适合学术论文的专业流程图。

## 1. 从算法代码生成流程图

### 适用场景
当您已有完整的算法代码，需要将其转换为直观的流程图用于论文展示。

### 通用提示词
```
请根据以下算法代码生成一个用于数学建模论文的专业流程图：

【插入您的算法代码】

要求：
1. 流程图应清晰展示算法的主要步骤、决策点和循环结构
2. 使用标准的流程图符号（开始/结束、处理、判断、输入/输出、连接等）
3. 步骤描述应简洁明了，使用学术性语言
4. 流程图布局合理，流向清晰，避免交叉线
5. 适合在A4纸大小的论文中使用，字体大小适中
6. 突出算法的核心创新点或关键步骤
7. 配色方案专业、简洁，避免过于鲜艳的颜色
8. 请添加适当的标题，概括算法的主要功能
```
