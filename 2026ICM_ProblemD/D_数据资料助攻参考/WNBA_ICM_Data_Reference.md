# 十、数据缺口与假设建议

## 10.1 数据缺口

| 缺失数据 | 重要性 | 替代方案 |
|----------|--------|----------|
| 球队详细财务报表 | 高 | 使用Sportico/Forbes估计 |
| 球员商业价值量化 | 中 | 社交媒体粉丝+代言合同估算 |
| 伤病历史数据库 | 中 | 使用NBA年龄-位置分布 |
| 票价分布数据 | 中 | SeatGeek/StubHub采样 |
| 运营成本结构 | 高 | 假设收入的70-80%为成本 |

## 10.2 假设建议

**财务假设:**
- 联盟年亏损: ~$40-50M (整体)
- 盈利球队: 估计3-5支 (Aces, Liberty, Fever)
- 平均运营成本: 收入的80-90%

**增长假设:**
- 收入年增长: 20-30% (近期), 10-15% (长期)
- 估值年增长: 15-25%
- 薪资帽增长: 新CBA后可能翻倍
